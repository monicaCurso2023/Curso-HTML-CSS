Selectores agrupados en CSS

Utilizamos selectores agrupados cuando queremos aplicar un estilo en particular a más de un selector

    --- CSS

    h1,
    h3,
    h5 {
      color:blue;
    }

    h2,
    h4,
    h6 {
      color:crimson;
    }

    ol,
    ul {
      font-style: italic;
    }

    .text-blue {
      color:blue;
    }

    .text-red {
      color:red;
    }

    .text-green {
      color:green;
    }

    --- HTML
    <h2>Título H2</h2>
  
    <h3>Título H3</h3>
    
    <ol class="text-green">
      <li>Pan</li>
      <li>Queso</li>
    </ol>
    
    <ul>
      <li>Vino tinto</li>
      <li>Vino blanco</li>
    </ul>
